# vscode再インストール後

## 拡張一覧

`code --list-extensions`

## 拡張インストール

`code --install-extension <extension_id>`


## 拡張インストールコマンド


### グローバル環境
```bash
code --install-extension mosapride.zenkaku # 全角文字のサポート
code --install-extension ms-ceintl.vscode-language-pack-ja # 日本語言語パック
code --install-extension editorconfig.editorconfig # EditorConfigのサポート
code --install-extension oderwat.indent-rainbow # インデントの可視化
code --install-extension shardulm94.trailing-spaces # 末尾の空白文字の削除
```

### JavaScript開発
```bash
code --install-extension christian-kohler.npm-intellisense # npmパッケージの補完機能
code --install-extension dbaeumer.vscode-eslint # ESLintの統合
code --install-extension denoland.vscode-deno # Deno開発サポート
code --install-extension esbenp.prettier-vscode # Prettierフォーマッタ
code --install-extension ms-playwright.playwright # Playwrightテストフレームワーク
code --install-extension vitest.explorer # Vitestテストランナー
code --install-extension hollowtree.vue-snippets # Vue.jsのスニペット
code --install-extension vue.volar # Vue.js開発サポート
```

### Git関連
```bash
code --install-extension eamodio.gitlens # Gitの詳細なビューと機能
code --install-extension mhutchie.git-graph # Gitのグラフィカルビュー
```



### Python開発
```bash
code --install-extension ms-python.debugpy # Pythonデバッグ
code --install-extension ms-python.python # Python開発サポート
code --install-extension ms-python.vscode-pylance # Pythonの型チェックと補完
```

### Java開発
```bash
code --install-extension redhat.java # Java開発サポート
code --install-extension vscjava.vscode-java-debug # Javaデバッグ機能
code --install-extension vscjava.vscode-java-dependency # Java依存関係管理
code --install-extension vscjava.vscode-java-pack # Java開発パック

code --install-extension vscjava.vscode-maven # Mavenプロジェクト管理

```
